
# 오노추 (오늘의 노래 추천)

## Overview

'오노추'는 "오늘의 노래 추천"의 줄임말로, 사용자의 현재 심리 상태를 10가지 질문을 통해 진단하고 그 결과에 가장 잘 어울리는 음악을 추천해 주는 감성 기반 음악 큐레이션 웹 애플리케이션입니다.

## Features & Design

*   **심리 진단 테스트**:
    *   10개의 질문으로 구성된 5점 척도 설문조사.
    *   사용자의 답변을 분석하여 6가지 주요 감정(기쁨, 슬픔, 당황, 상처, 불안, 분노) 중 가장 강한 감정을 도출.
*   **음악 추천**:
    *   각 감정마다 20곡씩, 총 120곡의 방대한 플레이리스트 보유.
    *   매 테스트마다 랜덤하게 1곡을 추천하여 매번 새로운 발견을 제공.
*   **디자인**:
    *   **오노추** 메인 타이틀과 **오늘의 노래 추천** 서브 타이틀을 중앙에 배치하여 명확한 서비스 아이덴티티 전달.
    *   세련된 글래스모피즘(Glassmorphism)과 감정별 다이내믹 컬러 테마 적용.
    *   모바일과 데스크톱 모두에 최적화된 반응형 UI.

## Technical Details

*   **Frontend**: HTML5, CSS3 (Custom Properties, OKLCH), Vanilla JavaScript.
*   **Deployment**: GitHub를 통한 배포.
